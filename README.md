# thesis - SignPic (Sistema Móvel Para Deteção de Língua Gestual Utilizando Machine Learning)

Para execução do ficheiro notebook de python utilizou-se o [Jupyter](https://jupyter.org/).

A biblioteca utilizado para implementação foi o [PyTorch](https://pytorch.org/). Para isso é necessário instalar o mesmo, para isso basta seguir as [instruções](https://pytorch.org/get-started/locally/) consoante o sistema operativo.

A implementação passo-a-passo está presente no Capítulo 3 da dissertação.

O conjunto de dados utilizado é proveniente do [Kaggle](https://www.kaggle.com/amarinderplasma/alphabets-sign-language) e é referente à Língua Gestual Americana.

---

Em relação à aplicação iOS esta é baseada na [aplicação demo iOS](https://github.com/pytorch/ios-demo-app) do PyTorch.

Para executar a aplicação é necessário um sistema com macOS e um iPhone caso se pretenda utilizar a camara como fonte para imagens de entrada.

Antes da execução é necessário executar o comando pod install para que as dependências da aplicação sejam instaladas.
